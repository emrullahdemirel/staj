## 📚 Projeler

### 1. 🧮 Hesap Makinesi
Temel matematiksel işlemler yapabilen basit hesap makinesi uygulaması.

**Özellikler:**
- Dört temel matematiksel işlem (+, -, ×, ÷)
- Hata kontrolü ve kullanıcı dostu mesajlar
- Enter tuşu ile hesaplama
- Responsive tasarım

**Teknolojiler:** HTML, CSS, JavaScript

📁 [Projeyi İncele](./hesap-makinesi/)

---

### 2. 🎨 Renk Değiştirici
Butona tıklandığında sayfanın arka plan rengini rastgele değiştiren uygulama.

**Özellikler:**
- Rastgele gradient arka plan renkleri
- Smooth geçiş animasyonları
- Space tuşu kısayolu
- Mevcut renk kodunu gösterme
- Başarı mesajı bildirimleri

**Teknolojiler:** HTML, CSS, JavaScript

📁 [Projeyi İncele](./renk-degistirici/)

---

### 3. 📱 Toggle Sidebar Menü
Modern ve responsive sidebar navigasyon menüsü.

**Özellikler:**
- Mobil ve desktop responsive tasarım
- Smooth açılma/kapanma animasyonları
- Alt menü (submenu) desteği
- Tema değiştirme (açık/koyu/otomatik)
- Klavye kısayolları (Ctrl+B, Alt+T, ESC)
- Local storage ile ayar kaydetme
- Kullanıcı bildirimleri

**Teknolojiler:** HTML, CSS, JavaScript, Font Awesome

📁 [Projeyi İncele](./toggle-sidebar/)

---

### 4. 🛒 Alışveriş Sepeti Uygulaması
Firebase Authentication entegrasyonlu e-ticaret sepet uygulaması.

**Özellikler:**
- Ürün listesi ve dinamik filtreleme
- Sepet yönetimi (ekleme, çıkarma, miktar değiştirme)
- Kullanıcı kayıt/giriş sistemi
- LocalStorage ile veri saklama
- Arama fonksiyonu
- Toplam hesaplama ve ödeme simülasyonu
- Responsive tasarım

**Teknolojiler:** HTML, CSS, JavaScript, Firebase Auth (demo), LocalStorage

📁 [Projeyi İncele](./alisveris-sepeti/)

---

### 5. ❓ FAQ Akordiyon Menü
Sık sorulan soruları gösteren interaktif akordiyon menü.

**Özellikler:**
- Smooth açılma/kapanma animasyonları
- Accessibility desteği (ARIA attributes)
- Klavye navigasyonu (Enter, Space, ESC)
- URL hash desteği
- Tek seferde bir soru açık kalma
- Responsive tasarım
- Tracking ve analytics desteği

**Teknolojiler:** HTML, CSS, JavaScript

📁 [Projeyi İncele](./faq-akordiyon/)

---

### 6. 🌤️ Hava Durumu Uygulaması
OpenWeatherMap API kullanarak hava durumu bilgisi gösteren uygulama.

**Özellikler:**
- OpenWeatherMap API entegrasyonu
- Detaylı hava durumu bilgileri (sıcaklık, nem, rüzgar, basınç vb.)
- Güneş doğumu/batımı bilgileri
- Son aramalar geçmişi (LocalStorage)
- Demo veriler ile çalışma
- Klavye kısayolları (Ctrl+L, ESC)
- Responsive tasarım ve loading animasyonları

**Teknolojiler:** HTML, CSS, JavaScript, OpenWeatherMap API, TypeScript tip tanımlamaları

📁 [Projeyi İncele](./hava-durumu/)

---

### 7. 🎨 Sanat Tarihi Portfolyo Sitesi
Next.js 14, TypeScript ve Supabase ile geliştirilmiş profesyonel portfolyo sitesi.

**Özellikler:**
- Modern ve responsive tasarım
- Next.js 14 App Router
- TypeScript tip güvenliği
- Tailwind CSS styling
- Supabase backend entegrasyonu (hazır)
- Loading animasyonları ve smooth transitions
- SEO optimizasyonu ve meta tags
- Accessibility desteği (WCAG)
- Contact form ve portfolio filtreleme
- Dark/Light mode hazır altyapı

**Bileşenler:**
- Header: Responsive navigation
- Hero: Animated hero section
- About: Tabbed content ve skill bars
- Projects: Grid layout ve modal detaylar
- Contact: Form ve sosyal medya bağlantıları
- Footer: Comprehensive footer

**Teknolojiler:** Next.js 14, TypeScript, Tailwind CSS, Supabase, React

📁 [Projeyi İncele](./portfolyo-sitesi/)

**Canlı Demo**: Projeyi çalıştırmak için:
```bash
cd portfolyo-sitesi
npm install
npm run dev
```
